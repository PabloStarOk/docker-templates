# docker-templates
Docker templates for `docker-compose.yaml`, `.devcontainer`, and `Dockerfile`.

## Current Templates

### Compose
- MySQL
- NodeJS
- Odoo
- PostgreSQL

### DevContainer
- A generic and personal `.devcontainer` configuration designed to be used with Docker Compose files.

### Images
- None available at the moment.
